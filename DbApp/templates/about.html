Smart DustBin

About

Online Dust Bin system after login to this app user can see nearly located Dustbins it's to ease user to put dust in the dustbin. When user puts some dust the automatic QR code generator screen is fixed on top of dustbin, he/she can scan the QR code and then they will get reward for using dustbin For Partial Use we Implemented this in my college this app admin can log in & visualize the content in the dustbin & inform to municipality officials. In a few days, I am going to give this prototype to government officials. This project included HTML, CSS and JavaScript are used in the front-end development of the application, and the back-end is built in the Django framework. Google Maps API is used to locate smart dustbins in real-time. The web application is converted into a mobile app using PhoneGap
